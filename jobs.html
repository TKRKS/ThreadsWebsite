Computer Science Jobs

Algorithm Researcher

Algorithms Researchers are the guys who come up with all the clever tricks that people use to make a program do a specific task quickly and efficiently. The develop, test and prove their methods will do what is needed in a short time and using little space if necessary. Algorithm Researchers will focus on the Theory Thread. The Theory Thread will teach you the math necessary not only to develop new algorithms, but also the math needed to prove the will work and work well.

Related Threads:

  • Theory

Compiler Writer

Without programming languages, it would be so incredibly difficult to program for computers that they very likely wouldn’t be in widespread use today. Compiler Writers are the people that translate from the high level programming languages that allow people to do so much to the low level instructions that the computer can actually understand. Compiler Writers need the Systems and Architecture and Theory Thread to do this. The Systems and Architecture Thread will help you understand computer hardware and how to actually tell the computer directly to do things using low level instructions. The Theory Thread is necessary to be able to figure out how actually translate from the high level programming levels to the low level instructions using many different kind of state machines.

Related Threads:

  • Systems and Architecture
  • Theory

Computer Animator

Computer Animators use computers to create animations that people watch for their entertainment. From children’s cartoons to CGI used in movies to cutscenes in video games, Computer Animators can create these animations for everyone to use. The Media and People Threads are the most useful ones for a Computer Animator. The Media Thread will teach you how to use computer graphics and how to actually animate things on the computer screen. The People Thread will teach you how to create visuals that are appealing to people so that they will enjoy watching your animations.

Related Threads:

  • Media
  • People

Computer Architect

Computer Architecture is the process of designing computer hardware and how they run. A Computer Architect is the person who actually plans out how hardware will run on a computer and the instructions that can be used to run them. Since a Computer Architect works so closely with computer hardware, they will find the Devices and Systems and Architecture Threads most useful. The Devices Thread has many computer engineering classes that will teach you about how computer hardware works and is built, which will be needed to make design decisions. The Systems and Architecture Thread will teach you how this hardware will interact with the software being run on it, so that the Computer Architect can make design decisions that will allow the computer to run as smoothly as possible.

Related Threads:

  • Devices
  • Systems and Architecture

Database Administrator

Database Administrators set up and run databases for various companies and agencies. They spend their time designing and implementing the logical database layout (like the tables and schemas) as well as managing the back end to ensure the database runs as quickly and efficiently as possible. This job lends itself to two Threads- Information Internetworks and Systems and Architecture. In Information Internetworks, there are multiple classes that focus completely on the logical database layout as well as the back end part of the database. As database efficiency deals directly with the operating systems and the underlying computer architecture, the Systems and Architecture Thread will give you the background needed to run a database as efficiently as possible.

Related Threads:

  • Information Interworks
  • Systems and Architecture

High Frequency Trader

High Frequency Traders are people who use computers to make trades on the stock market so that even a fraction of a penny increase in price is enough to make thousands of dollars in a fraction of a second. These people need to write programs that can predict the stock market and quickly react to that predictions. The Modeling and Simulation Thread is ideal for this job. In Modeling and Simulation, you will learn how to create and program models which can be used to help predict changes in the stock market. In addition, you will learn how to run these models on incredibly fast computers so that you can make these trades fast enough to earn your profit.

Related Threads:

  • Modeling and Simulation

Mobile Developer

Mobile Developers create programs that are run on phones and other small devices. Whether they are a simple calendar system or apps for popular websites, Mobile Developers must always focus on both the limited hardware they have to work with and the that that the phones are always on the move. Mobile Developers will find the Devices and the Information Internetworks Threads most useful. The Devices Thread teaches you how to write programs that are designed specifically to run on limited hardware. The Information Internetworks Thread will teach you how to stay connected to a mobile device that is always on the move (which is much harder than it sounds).

Related Threads:

  • Devices
  • Information Interworks

Network Administrator

Network Administrators set up and run large scale networks for different companies. They need to make the network run quickly, run efficiently and be resilient against all kinds of failure. The Devices Thread will help teach you how the hardware that connects to the networks actually works, so you can know what to expect when you add additional nodes to your network. In addition, the Information Internetworks Thread will teach you the networking you need in order know how to set up and run these networks in addition to teaching you security to help protect them.

Related Threads:

  • Devices
  • Information Interworks

Operating Systems Designer

Operating Systems are some of the largest and most complicated programs in existence. As such, working on them is a very specialized job. Operating Systems Designers develop and add on to operating systems that take care of the computer hardware so other programs don’t have to worry. The Systems and Architecture Thread is perfect for an Operating Systems Designer. This entire Thread is focused on computer hardware and its interaction with computer software (in fact, Operating System Design is one of the main classes for the Thread), as such it will be a great help for an Operating System Designer.

Related Threads:

  • Systems and Architecture

Roboticist

Roboticists design, build and program robots that do all kinds of different tasks. Whether they are working on simple robots that work on assembly lines or soccer playing robots, Roboticist must be familiar with the computer hardware necessary to build the robots as well as the algorithms needed for the robot to run on its own. The two most useful Threads for a Roboticist would be Devices and Intelligence. The Devices Thread will teach you about the computer hardware and how to write programs that work with very limited hardware (which is useful because robots may not be able to fit a full scale computer within them). The Intelligence Thread will teach you about algorithms which will allow the robot to work autonomously without input from the Roboticist.

Related Threads:

  • Devices
  • Intelligence

Security Specialist

Security Specialists are the heroes of the computer world. They work to stop hackers, virus and more from hurting people and their computers too much. Since hackers target a wide variety of computers and programs, Information Security Specialists must both understand how the internals of computers and the internet works. As such, the best Threads for this kind of job would be Information Internetworks and Systems and Architecture. In Information Internetworks, you will learn both the basics of information security that can protect information both on and between computers, and how the internet works so you are aware how viruses can transfer themselves. In addition, since virus often target bugs deep in the operating system code, the Systems and Architecture Thread will give you a good basis necessary to combat these attacks.

Related Threads:

  • Information Interworks
  • Systems and Architecture

Video Game Designer

The video game industry has become a multibillion dollar industry in the past decade. As such, they always need new people who can program these giant, complex games. Video Game Designers program these video games, often with their own specific specialization. Depending on your specific interests, a student who wants to be a Video Game Designer may find any of the Information Internetworks, Intelligence, People, and Media Threads useful. The Information Internetworks Thread will teach a student how to make two computers interact with each other, which is needed if you want to make a multiplayer game. The Intelligence Thread is great for learning how to program artificial intelligence for the characters in your game. The People Thread will teach you how to make easy to use user interfaces that will allow anyone to easily pick up and play your game. Finally, the Media Thread will teach you how to program computer graphics to make your game look nice. Depending on your specific end goal as a Video Game Designer, any of these Threads would give you foundation with which you can make your game.

Related Threads:

  • Information Interworks
  • Intelligence
  • Media
  • People

Web Designer

Web Designers create websites that people use every day. They have to both make a front end that is appealing and intuitive to users and create the back end interactions that will allow the site to run well on the internet. As such, Web Designers would find the Information Internetworks and People Threads. The Information Internetworks Thread will give you the knowledge of the internet necessary to create a back end to the website that will work with all the users who will view it online. The People Thread will teach you how to create intuitive and helpful user interfaces that will grab the user’s attention and keep them coming back for more.

Related Threads:

  • Information Interworks
  • People

Media

Computers have become a major source of entertainment for many different people. In the past few decades, people have been using computers more and more for fun rather than just business. From computer-animated movies and TV shows to interactive video games, there are many different ways someone can have fun with a computer. The Media Thread will teach you how to make something entertaining using computers.

The Media Thread is all about using computers to make some kind of entertainment for others. For example, you will learn how to create computer graphics and make animations on the computer screen. You may also learn about video game design and how to make something fun to play with a computer. You could also learn about computer audio and how the computer can make sounds. All this and more can be found in the Media Thread.

Many different kinds of people will like the Media Thread. Creative types will like the many different outlets for their ideas. Students who are interested in creating video games or making animated movies will find this Thread perfect for them. Fans of graphics and game engines will find themselves right at home with the programming that is done in this Thread.

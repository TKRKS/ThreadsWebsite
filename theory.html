Theory

When you look at the fundementals behind programming, you discover that behind all the crazy things that computers can do is math. Whether it is an interesting algorithm that manages to greatly speed up a program or a clever data structure that allows a program to work with limited memory, this all boils down to math. The Theory thread deals with this math behind programming and will teach students how to think up new methods to program various problems well. You may enjoy this thread if you like math and algorithms even more than programming itself.
